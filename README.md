# SkyNet
# Lista de Tarefas

## Objetivo
Aplicação simples para adicionar e remover tarefas, aplicando conceitos básicos de DevOps como versionamento e automação.

## Funcionalidades
- Adicionar tarefas
- Remover tarefas
- Validação de campo vazio

## Tecnologias
- HTML
- CSS
- JavaScript

## Integrantes
- Seu nome
- Nome do colega

## Como executar
1. Baixe o projeto
2. Abra o arquivo index.html no navegador
